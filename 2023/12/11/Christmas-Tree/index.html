Christmas Tree

It is a Christmas tree to celebrate Christmas and joy someone.

我一开始打算用 go + fyne 画个GUI的圣诞树，因为写好的程序可以打包很方便，发给别人看也可以直接得到效果，不需要配环境。但是学了几天，我发现 fyne不好用（这只是我的主观看法，没有任何其他意思），跟着官方文档学的时候遇到最多的问题就是fyne.io/fyne 和 v2 不兼容:

大概率是我太菜了，不懂怎么改。

后来我选择了前端，这个更方便。然后我去 Github 找大佬写的 demo， 没花多久就找到了一个我想要的，简直太棒啦！我在大佬的项目上修修改改，加了个根据当前时间来显示不同的画面的功能然后就直接拿来用了。我在修改注脚的时候发现 @Hao Chen ，本来没太在意，然后我加了个超链接跳转到原作者的仓库，我看到原仓库下有个网址，是酷壳 然后我看着原作者的头像觉得好像在哪见过，又看了看名字，不会是陈皓大佬吧？！然后我又想起注脚，哦，真的是那个 “左耳朵耗子” ！几个月前刚听到大佬因心梗去世的消息时回忆起我初学编程时老师推荐我们逛酷壳等一些技术博客学习，我学习指针时还是看 《C语言结构体里的成员数组和指针》解惑的，皓哥分析的很详细，不过当时的我看的很懵，因为刚学，不懂gdb，所以gdb那部分就略过去了…

没想到，我学编程的路上一直有皓哥的帮助。R.I.P
